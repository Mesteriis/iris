# Review Snapshots

This section groups review documents that were written at a specific point in time.

They are useful for architectural context, risk framing, and historical decisions, but they can lag the live repository.

## Available Reviews

- [Backend Business Architecture Review](backend-business-architecture-review.md)
- [Backend Technical Review](backend-technical-review.md)
- `deep-research-report (1).md` remains an archived working snapshot and is intentionally not treated as a navigation entrypoint

## Interpretation Rule

If a review conflicts with current repository behavior:

1. prefer generated docs in `docs/_generated/`
2. then prefer accepted architecture docs in `docs/architecture/`
3. then use the review as historical context only
